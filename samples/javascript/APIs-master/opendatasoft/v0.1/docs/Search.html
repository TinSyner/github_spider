Open Data Soft API: Search

The Search API offering powerfull full text search features to navigate across the records of a dataset.

search

Parameter name Description Default Optionnal
q full text search query (searches over dataset title, description and meta data). none false
format the output format, one of json, jsonp and csv json true
hf number of search results to retrieve none true
start id of the first result to retrieve (useful for implementing pagination) none true
