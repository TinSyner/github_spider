POST checkin

Allows you to check in to a place.

Default
Description
Parameter
string

The venue where the user is checking in. No venueid is needed if shouting or just providing a venue name. Find venue IDs by searching or from historical APIs.

venueId

Required

string

If are not shouting, but you don't have a venue ID or would rather prefer a 'venueless' checkin, pass the venue name as a string using this parameter. It will become an 'orphan' (no address or venueid but with geolat, geolong).

venue

Optional

string

A message about your check-in. The maximum length of this field is 140 characters.

shout

Optional

string

How much to broadcast this check-in, ranging from private (off-the-grid) to public,faceboook,twitter. Can also be just public or public,facebook, for example. If no valid value is found, the default is public. Shouts cannot be private.
