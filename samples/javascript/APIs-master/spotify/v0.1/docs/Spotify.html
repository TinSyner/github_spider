Spotify API

The Spotify Web API enables you to retrieve data from the Spotify service

Method Description Example
lookup(params) Looks up an artist, album or track uri. Using the extras parameter you can get more detailed information in the response, only for artist and album uris. 
apis.spotify.lookup('spotify:artist:4YrKBkKSVeqDamzBPWVnSJ')
or
apis.spotify.lookup({uri: 'spotify:artist:4YrKBkKSVeqDamzBPWVnSJ', extras: 'album'})
or
apis.spotify.lookup({uri: 'spotify:artist:4YrKBkKSVeqDamzBPWVnSJ', extras: 'albumdetail'})
or
apis.spotify.lookup({uri: 'spotify:album:6G9fHYDCoyEErUkHrFYfs4', extras: 'track'})
or
apis.spotify.lookup({uri: 'spotify:album:6G9fHYDCoyEErUkHrFYfs4', extras: 'trackdetail'})
searchAlbum(params) Searches for an album. Use the q attribute or params as the query string and page as the page of the result set to return. 
apis.spotify.searchAlbum('plastic beach')
or
apis.spotify.searchAlbum({q:'are you experienced', page: 2})
searchArtist(params) Searches for an artist. Use the q attribute or params as the query string and page as the page of the result set to return. 
apis.spotify.searchArtist('Them Crooked Vultures')
or
apis.spotify.searchArtist({q:'john', page: 2})
searchTrack(params) Searches for a track. Use the q attribute or params as the query string and page as the page of the result set to return. 
apis.spotify.searchTrack('Master of Puppets')
or
apis.spotify.searchArtist({q:'GANGNAM STYLE', page: 2})
player(params) Creates a play button for the given uri(s). The uri parameter can either be a string representing a single uri or an array containing multiple track uris, in that case the play button would be a trackset whose title can be chosen. You can set the button's height, width, theme (black or white) and view(list or coverart). 
apis.spotify.player('spotify:track:4NFhtFE68cXUrfbne8JE2c')
or
apis.spotify.player({uri: 'spotify:album:3BTvFnJt1Hhh985s3oZ61L', width: 800, height: 600, theme: "white", view: "list"})
or
apis.spotify.player({uri: ['spotify:track:5JunxkcjfCYcY7xJ29tLai', 'spotify:track:40lY9BHMh2bFE5aFIJyGgy'], width: 800, height: 150})
