Google Chart API

Initialisation

To initialize a google chart : apis.google.chart(options)

options is a JSON which contains the following key

Parameter Description Default Optionnal
datasource The datasource of the maps No default value false
columns If datasource is not set, it will process columns and data to generate the chart No default value false
data The data of the chart (used with columns) No default value false
type The type of the chart. It can be piechart, linechart, areachart. piechart false
title The title of the chart By default, no title is set, title is a string false
width The width of the chart 300 false
height The height of the chart 300 false
bgcolor The background color of the chart #FFF false

e.g : var chart = apis.google.chart({datasource: [["a", 38], ["b", 75]], title: "My custom chart"})

Manipulation

Once your Google Chart instanciated. You can manipulate it using the object stored in chart variable

Google Chart has the following interface :

Method Description Exemple
resize(width, height) resize the chart with width and height 
chart.resize(800, 300);
