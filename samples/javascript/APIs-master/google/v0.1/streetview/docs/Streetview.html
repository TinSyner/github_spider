Streetview Image API

The Google Street View Image API lets you embed a static (non-interactive) Street View panorama or thumbnail into your web page, without the use of JavaScript. The viewport is defined with URL parameters sent through a standard HTTP request, and is returned as a static image.

Method Description Exemple
streetview(opts) Retrieves the streetview image of the given location opts.location or latitude/longitude couple opts.latitude opts.longitude. You can specify the image's size opts.size, the field of view opts.fov, heading, pitch, sensor. apis.google.streetview({location: "palais brogniart", pitch: 10, heading: 235, fov: 90, sensor: false})
