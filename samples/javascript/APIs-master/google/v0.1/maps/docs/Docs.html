Google Maps API

Integrating Google Maps with Webshell is pretty easy

Initialisation

To initialize a google map : apis.google.maps()

Some parameters can be given to maps()

Parameter Description Default Optionnal
width The width of the map Container width false
height The height of the map 300 false

e.g : apis.google.maps({width: "550px", height: "400px"})

Manipulation

Once your Google Map is instanciated : var m = apis.google.maps();. You can manipulate it using the object stored in the m variable

Google Maps has the following interface :

Method Description Exemple
center(location) Centers the google map on location, which can be an address or a latitude, longitude couple. 
m.center('paris');
zoom(level) Sets the Zoom level for your Google Map. zoom is a number between 1 and 18 
m.zoom(16);
addMarker(location) Adds a marker on the Google Map at a specified location, which can be an address or a latitude, longitude couple. This marker can have a title and an icon 
m.addMarker('Palais brongniart');
or
m.addMarker({
    address: "San Francisco, Fort Mason",
    icon: "http://www.perigorddecouverte.com/marker_fermier.png",
    title: "Would be nice to be there"
});
streetview(location) Gets the panoramic 360 degree Street View of the location, which can be an address or a latitude, longitude couple. The map's current center position si used by default. 
m.center('9 place de la Bourse');
m.streetview();
or
m.streetview('9 place de la Bourse);
directions(opts) Calculates directions between origin and destination. You can set the travelMode to either walking, driving, bicycling or transit. With the draggable option set to true, the direction can be modified afterwards 
m.directions({origin: "9 place de la bourse", destination: "eiffel tower paris", travelMode: "walking", draggable: true});
