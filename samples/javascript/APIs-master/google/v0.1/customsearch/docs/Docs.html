Google API: Custom Search

These methods are relative to the tag 'Custom Search'

search

Retrieves results for a particular search. Custom search engine identifier: Use either cx or cref to specify the custom search engine you want to perform this search. Use cx for a search engine created with the Google Custom Search page. Use cref for a linked custom search engine. If both are specified, cx is used.
