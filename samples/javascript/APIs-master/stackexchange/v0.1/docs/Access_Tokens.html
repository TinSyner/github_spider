GET accessTokensGet

Reads the properties for a set of access tokens.

Default
Description
Parameter
string

{accessTokens} can contain up to 20 access tokens. These are obtained by authenticating a user using OAuth 2.0.

accessTokens

Required

integer

The pagination offset for the current collection. Affected by the specified pagesize.

page

Optional

integer

The number of collection results to display during pagination. Should be between 0 and 100 inclusive.

pagesize

Optional
