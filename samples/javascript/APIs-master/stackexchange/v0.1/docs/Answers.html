GET answersCommentsGet

Gets the comments on an answer.

A single primary key identifier or a vectorized, semicolon-delimited list of identifiers. This is a vectorized parameter, multiple ids can be sent a single request if delimited with a semi-colon string. {id} can contain up to 100 semicolon delimited ids, to find ids programatically look for answer_id on answer objects.
