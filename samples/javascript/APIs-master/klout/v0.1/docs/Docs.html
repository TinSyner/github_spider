Klout API

Method Description Exemple
getKloutId(params) This method allows you to retrieve a KloutID from numeric twitter id or google+ id. The params.userId is the google+ or twitter id and params.idType can be tw, twitter, gp or google+. 
apis.klout.getKloutId({userId: "112620727907435773834", idType: "google+"})
getTwitterId(kloutId) This method allows you to retrieve a TwitterId for a numeric KloutId. 
apis.klout.getTwitterId('45598950992084523')
score(kloutId) This method allows you to retrieve a user's Klout Score and deltas. 
apis.klout.score('635263')
showUser(kloutId) This method allows you to retrieve a user. 
apis.klout.showUser('635263')
topics(kloutId) This method allows you to retrieve a user's topics. 
apis.klout.topics('635263')
influence(kloutId) This method allows you to retrieve a user's influencers and influencees. 
apis.klout.topics('635263')
