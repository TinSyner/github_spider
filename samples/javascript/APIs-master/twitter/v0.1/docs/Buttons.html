Twitter Button Widgets

Method Description Example
tweetButton Displays a tweet button for the given url. The button options are listed bellow 
apis.twitter.tweetButton("http://webshell.io")
or
apis.twitter.tweetButton({url: "http://webshell.io", size: "large"})
followButton Displays a follow button for the given twitter screen_name. The button options are listed below 
apis.twitter.followButton("Webshell_")
or
apis.twitter.followButton({screen_name: "Webshell_", "data-show-count": "false"})

Here are the options for the tweet button

Tweet Button Option Description
url URL of the page to share
via Screen name of the user to attribute the Tweet to
text Default Tweet text
related Using the related field you can suggest accounts for a user to follow once they have sent a Tweet using your Tweet Button. These suggested accounts and their basic information are shown on the last page of the Share Box flow. Only two accounts are displayed and by default the via user is shown first with the first related account shown afterwards. If the user is a follower of the via user the Share Box will instead show the first two related accounts the user isn't a follower of. No accounts are displayed if the user follows all of the suggested accounts (via and related). You can add your own summary of a related user by adding some text after their screen name, separated using a colon. For example, to add a summary The Javascript API to the related user @anywhere you would use: related:"anywhere:The Javascript API". The summary is shown above the related user and is in addition to the default information like the bio and verified status. Summaries should not include commas or colons and can only be added to related accounts.
count The count box shows how many times the URL has been Tweeted. You can choose to display or hide the count box, or place it above or next to the Tweet Button. When no value is given for the count box the Tweet Button will default to horizontal. Values for the count property can be none, horizontal or vertical
lang The language for the Tweet Button
counturl URL to which your shared URL resolves
hashtags Comma separated hashtags appended to tweet text
size The size of the button can render in either medium, which is the default size, or in large - which is the larger button.
dnt Twitter buttons on your site can help us tailor content and suggestions for Twitter users. If you want to opt-out of this feature, set the optional dnt parameter to be true.
Tweet Button Option Description
data-show-count By default, the User's followers count is displayed with the Follow Button. You can disable the display of this number using the data-show-count property
data-lang The Follow Button is available in English (en), French (fr), German (de), Italian (it), Spanish (es), Korean (ko) and Japanese (ja).
data-width You can specify the width of the Follow Button using the data-width (in pixels or percentage) parameter
data-align You can specify the alignment of the Follow Button using the data-align (left or right) parameter.
data-show-screen-name The user's screen name shows up by default, but you can opt not to show the screen name in the button. You can do this by using the data-show-screen-name attribute.
data-size The size of the button can render in either medium, which is the default size, or in large - which is the larger button.
data-dnt Twitter buttons on your site can help us tailor content and suggestions for Twitter users. If you want to opt-out of this feature, set the optional data-dnt parameter to be true.
