POST classifytext
Default
Description
Parameter
string

The two letter language code of the text you wish to submit. Accepted values are:"en" - English, "es" - Spanish", pt" - Portuguese, "fr" - French, "nl" - Dutch, "de" - German, "it" - Italian. For ULTRA users with detectlang set to 1, this lang parameter will only be used as a fall back if automatic language detection fails.

lang

Optional

string

The text you wish to be classified. The maximum length is 300 characters. This string must be UTF-8 encoded (otherwise non-ascii characters will not be recognised properly).

text

Required

string

Any words you wish to exclude to avoid swaying the sentiment detection. This may be the name of a product for example. Enter a comma separated string.

exclude

Optional

string

For subscribers of the ULTRA plan only, when this value is set to 1 Chatterbox will automatically detect the status language. If it is unable to automatically detect the language, Chatterbox will fall back on the lang parameter.
