## WShellV

### 用法

```bash
[~]$ git clone https://github.com/akkuman/WShellV.git

[~/WShellV]$ cd WShellV

[~/WShellV]$ pip install pipenv

[~/WShellV]$ pipenv install

[~/WShellV]$ pipenv shell

[~/WShellV]$ python app.py

```
然后打开 `http://127.0.0.1:8080` 即可
